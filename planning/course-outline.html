DADS 5250 — Generative AI in Practice

Graduate Course | Northeastern University | Department of MIE
Instructor: Mohammad Dehghani, Ph.D.
15 Modules (M00 - M14) 4 Phases + Intro + Hackathon
This graduate course teaches applied Generative AI -- from LLM API basics to production-ready agent systems. Students build hands-on with the latest SDKs (OpenAI, Claude Agent SDK, Google ADK), learn prompt engineering, RAG, multi-agent orchestration, MCP, and workflow automation. The course emphasizes practical skills over theory (20/80 split) with Jupyter notebook labs on Google Colab. Each module features an AI Spotlight -- a real-world AI tool students should know.
