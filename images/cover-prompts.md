# DADS 5250 — Module Cover Image Prompts (v2)

> Ready to batch-generate with NanoBanana MCP (`/gen-image`)
> **Aspect ratio**: `4:3` (set first with `set_aspect_ratio`)
> **Output**: `images/M##-cover.png`
> **Style reference**: Edge1.ai course-v2 + bootcamp-sec images

## Style Guide

All covers share this **Edge1-inspired** aesthetic:
- **Background**: Pure black / very dark charcoal — NOT navy, almost black
- **Composition**: 3D isometric floating platform OR central object with floating glass panels
- **Lighting**: Dramatic studio lighting from above/behind, strong reflections on dark surfaces
- **Glass UI elements**: Floating translucent dashboard cards, code snippets, diagram panels
- **Accent color**: Each module gets a unique color that appears IN the content (charts, node colors, UI highlights, platform edge glow)
- **Feel**: Premium, cinematic, Apple-keynote style, photorealistic 3D render
- **NO text, NO logos** — text is added separately
- **Quality**: Ultra high resolution 4K, photorealistic 3D render, octane render style, studio lighting

## Batch Generation Steps

When NanoBanana MCP is connected:

1. `set_aspect_ratio` → `4:3`
2. For each module, call `gemini_generate_image` with the prompt → save to `images/M##-cover.png`
3. Add 3s delay between generations
4. Open `cover-review.html` to compare results
5. Re-generate any that need tweaking
