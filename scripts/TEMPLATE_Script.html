Let me show you what an API call actually looks like. Seven lines of Python.

from openai import OpenAI

client = OpenAI()
response = client.chat.completions.create(
    model="gpt-4.1-mini",
    messages=[{"role": "user", "content": "Hello"}]
)
print(response.choices[0].message.content)

That is it. That is the entire program. Seven lines. Let me walk you through each one.
