Video 2: Introduction to n8n

Voiceover + screen recording -- 6-8 minutes -- 3 segments
SEG A What Is n8n
Voiceover + Screen ~3 min

n8n is a workflow automation platform. Think of it as a visual programming environment where each node is an action -- send an email, call an API, query a database, process text with AI.

You connect nodes with wires. Data flows from one node to the next. When something triggers the workflow -- a new email, a scheduled time, a webhook -- the entire pipeline runs automatically.

Demo: n8n Canvas
Screen Recording The actual n8n workflow builder -- nodes connecting with wires. Email trigger, AI processing node, database node, notification node. Visual, no-code, intuitive. ~20 sec
MEDIA TEAM: Record the n8n canvas in action. Show nodes being dragged onto canvas, wires connecting them, data flowing through on execution. Highlight the visual nature -- no code visible, just connected nodes. Use a simple 3-node workflow for clarity.

Now that you have the basics, let us connect LLMs to n8n workflows.

SEG B Three Production Patterns
Voiceover + Screen ~4 min

I will show you three patterns. First, document processing -- a file arrives, AI extracts structured data, it goes into a database. Second, intelligent routing -- an email comes in, AI classifies it, and the workflow routes it to the right team. Third, content generation -- a scheduled trigger fires, AI generates a report from database data, and it gets emailed to stakeholders.

These are not toy examples. These are the exact patterns businesses use in production today.
