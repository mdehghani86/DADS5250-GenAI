PART 1: RECAP
------------------------

You just gave your AI agent the ability to reach into real systems.

You built an MCP server that exposes tools through a standard protocol. You connected a model to it and watched it discover and use those tools on its own. And you built guardrails that define the boundaries -- what the agent can do, what it cannot, and what requires human approval.

This is the difference between a toy and a product. Connection plus control.

[ANIMATION: Brief recap -- MCP server diagram, agent calling a tool, guardrail checkpoint approving an action]

PART 2: WHAT IS NEXT
------------------------

Your agents can now read, reason, act, coordinate, and connect to external systems. But so far, everything has been text.

In Module 10, we give your agents eyes. Vision APIs. Multimodal input -- text plus images. And then we ask the critical question: how do you actually measure if any of this is working? Evaluation.

Your agents are about to see the world. And you are about to learn how to grade them.
