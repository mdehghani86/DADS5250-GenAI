Six modules of skills. APIs, tools, prompting, structured output, chains, RAG, agents. Now put them all together.

Build something real.

Not a homework assignment. Not a lab exercise. A product. Something you could demo to a hiring manager, show to a client, or pitch to an investor.

Every tool you have learned is fair game. OpenAI, Gemini, LangChain, RAG pipelines, agents -- mix and match. The only rule is that your project must use at least two techniques from the course.

Teams of two or three. Pick people whose skills complement yours. If you are strong on the backend, find someone who can build a compelling demo. If you are great at prompting, pair with someone who can wire up the infrastructure.

The best hackathon projects I have seen were not the most complex. They were the most focused. One clear problem, one clear solution, executed well.

Build something real. Break something. Ship it.

Welcome to the hackathon. Let me walk you through how this works.

First, teams. You should already be in teams of two or three. If you are not, raise your hand now and we will match you.

Timeline. You have from kickoff until the submission deadline. My advice: spend the first chunk of time planning. Decide what you are building, sketch the architecture, divide the work. Do not touch code until you have a plan.

Submissions. You will submit a GitHub repository with a README, a working demo, and a three-minute video walkthrough. The video is not optional -- it is how we judge your project.

Judging. Four criteria, equal weight: technical depth, creativity, polish, and presentation. Technical depth means you used course techniques meaningfully -- not just an API call, but something with structure. Creativity means you picked an interesting problem. Polish means it works without crashing. Presentation means you can explain it clearly in three minutes.

Office hours will be running continuously. Use them. Ask questions early. Do not wait until the last minute to discover your pipeline is broken.

One more thing. Have fun. This is the most free you will be in this course. Use it.

Hackathon Guide

Everything you need to plan, build, and submit a winning project.

Project Ideas

Not sure what to build? Here are categories that have produced strong projects in past cohorts:

  • AI Assistant: A domain-specific chatbot with RAG -- legal research, medical FAQ, course advisor
  • Content Pipeline: A chain that takes raw input and produces polished output -- blog posts, reports, social media threads
  • Data Analyzer: Upload a CSV, ask questions in plain English, get charts and insights
  • Code Tool: An agent that reads a codebase and answers questions, writes tests, or generates documentation
  • Workflow Automator: Connect multiple APIs to solve a real business process -- email summarizer, meeting prep, competitive analysis

Judging Criteria

CriterionWeightWhat We Look For
Technical Depth25%Meaningful use of 2+ course techniques. Not just an API call -- show architecture.
Creativity25%Original problem choice. Surprising solution. Something we have not seen before.
Polish25%It works. No crashes. Clean code. Good error handling.
Presentation25%Clear 3-minute video. You can explain what you built and why it matters.

Technical Requirements

  • Python-based (Jupyter notebook or standalone script)
  • Must use at least 2 techniques: API calls, prompt engineering, structured output, LangChain, RAG, or agents
  • Working demo -- the judges will run your code
  • README with setup instructions, architecture diagram, and team member contributions

Resources

  • All course labs and notebooks -- reuse and extend freely
  • OpenAI and Gemini API keys (your existing keys)
  • Office hours running continuously during the sprint
  • Slack channel for team coordination and quick questions

AI Startup Inspiration

Four companies that started with the same building blocks you have. Your hackathon project could be the seed of something like this.

Jasper AI
Content Generation
Founded in 2021 by Dave Rogenmoser, Chris Hull, and John Philip Morgan. Built on GPT-3 API calls to generate marketing copy. Reached $80M ARR by 2023. Started as a simple prompt-to-content tool -- exactly the kind of pipeline you could build in a hackathon.
Midjourney
Image Generation
Founded in 2022 by David Holz, former co-founder of Leap Motion. Operated entirely through Discord with no traditional UI. Grew to an estimated $200M+ annual revenue by 2023 with fewer than 40 employees. Proved that a creative interface around a model can be a billion-dollar business.
Cursor
AI-Powered Coding
Founded in 2023 by Aman Sanger, Sualeh Asif, Michael Truell, and Arvid Lunnemark. An AI code editor built on top of LLM APIs. Raised $60M Series A in 2024. The core idea: wrap an LLM in a developer-friendly interface with context from the codebase -- structured output plus RAG.
Perplexity
AI-Powered Search
Founded in 2022 by Aravind Srinivas, Denis Yarats, Johnny Ho, and Andy Konwinski. Combines web search with LLM summarization -- a RAG pipeline at scale. Valued at $9B by late 2024. The architecture is essentially what you learned in the RAG module, applied to the entire internet.

"Every one of these companies started with the same tools you have right now -- an API key and an idea. The difference is they shipped it."

Submission Checklist

  • GitHub Repository -- Clean code, working demo, requirements.txt or environment setup
  • README -- Project description, architecture diagram, setup instructions, team member contributions
  • 3-Minute Video -- Demo the product, explain the architecture, highlight what you are proud of

Presentation Schedule

Presentations happen in the class session following the hackathon deadline. Each team gets 3 minutes to present plus 2 minutes for Q&A from judges. Order will be randomized. Be ready to demo live -- judges may ask you to show something not in the video.
