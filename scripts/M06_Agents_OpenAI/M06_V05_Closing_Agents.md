PART 1: RECAP
------------------------

Let us take a moment to appreciate what you just built.

You created an AI system that reasons. It looks at a problem, picks a tool, uses it, evaluates the result, and decides what to do next. That is not a chatbot. That is not a chain. That is an autonomous agent.

You used the OpenAI Agents SDK to define agents with tools, handoffs, and guardrails. You used tracing to watch exactly how the agent thinks through a problem — every decision, every tool call, every observation.

You now understand the ReAct pattern. You understand the difference between scripted workflows and autonomous reasoning. That is a fundamental shift in how you think about AI systems.


PART 2: WHAT'S NEXT
------------------------

Next module, we go wider.

You built agents with OpenAI's SDK. But OpenAI is not the only player. Anthropic has the Claude Agent SDK. Google has the Agent Development Kit. Each has different strengths, different design philosophies, different tradeoffs.

You are going to build with all of them. By the end of Module 7, you will be fluent across three major agent platforms — and you will know when to reach for each one.

See you in Module 7.
