PART 1: RECAP
------------------------

You just built agents with the two most powerful SDKs in the industry.

With Claude Agent SDK, you saw how an agent can read files, write code, search the web, and reason through complex tasks -- all from a single import. With Google ADK, you saw the same idea through Google's lens -- built-in search, code execution, and tight integration with Google Cloud.

You did not wire together five libraries. You did not write boilerplate. You told the agent what to do, gave it the right tools, and it figured out the rest.

That is what batteries-included means in practice.
